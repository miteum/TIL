# 문법 시작

## SELECT 

> 조회할 때 사용하는 구문 

```
SELECT * FROM 테이블명. (컬럼명)
```

*(asterisk) - 각 row의 모든 값을 조회 한다는 의미 

from ~ 로 부터 즉, 어느 데이터로 부터

특정 컬럼 이름을 써도 무방!!! 

----

WHERE은 특정 조건을 설정하기 위해 사용

```
SELECT * FORM 테이블명 WHERE email = 'kkk@kkkk.com'
```

email 컬럼에 있는 kkk@kkkk.com만 조회 할 수 있는 쿼리문임
